# Reference Documentation — Spanish Funds & Depositary Market

Curated reading list to understand the Spanish UCITS / pensions / depositary
landscape and reconcile data sources used in the Competitive Landscape memo.

## Files in this folder

| File | What it is | Use it for |
|---|---|---|
| `CNMV_Estadisticas_IIC_2025_Q4_Resumen.pdf` | CNMV official Estadísticas IIC, Q4 2025 (resumen) | Official regulatory totals for FI + SICAV + IIC Inmobiliaria + Foreign IIC marketed in Spain |
| `Cecabank_Resultados_2025_NotaPrensa.pdf` | Cecabank 2025 full-year results press release | Cecabank's own depositary AUC: €398.8B custody / €285.7B depositaría at FY25 |
| `INVERCO_Informe_Anual_IIC_y_Pensiones_2024.pdf` | INVERCO industry annual report (2024 data, published 2025) | Long-form context on Spanish IIC + pensions market structure |
| `INVERCO_Pensiones_2025Q4_NotaPrensa.pdf` | INVERCO Pensiones Q4 2025 press release | Official pensions totals: €137,988M total (Indiv. €96.3B + Empleo €40.9B + Asoc. €0.8B) |
| `Inversis_Depositaria_36pct_growth_NotaPrensa.html` | Inversis press release on +36% depositaría growth | Inversis's own narrative on its independent-asset-managers niche |

## Files NOT in this folder (and why)

- **CNMV Estadísticas IIC 2025 Q4 — Anexo (full statistical annex)**: 305 MB,
  too large to mirror locally. Download from
  https://www.cnmv.es/portal/Publicaciones/PublicacionesGN.aspx?id=27 if needed.
  The Excel files in `../Data/CNMV Estadisticas/` already contain the same data
  in machine-readable form.
- **FundsPeople IX Ranking de Depositaría (2026 edition, data 2025-12-31)**:
  registration-required — no public PDF. The headline figures (€693.5B total,
  +12% YoY, top-5 = 92%) are extracted from FundsPeople's open-access summary
  article. Inversis full breakdown and BNP Alternativos €23.85B were provided
  by the user from the printed/registered edition.

## How to read these together

1. **CNMV** = ground truth for regulated FI + SICAV depositary assignments
   (downloadable per-fund register).
2. **INVERCO Pensiones** = ground truth for pensions totals; pensions
   depositary is NOT public per fund — must be approximated from individual
   gestora disclosures.
3. **FundsPeople IX** = the only publication that consolidates depositary AUM
   across FI + SICAV + Pensiones + EPSV + Alternativos in one ranking.
4. **Cecabank own publications** = useful triangulation: Cecabank's €285.7B
   total is materially higher than what we can reconstruct from public
   sources (≈€268B), implying ≥45% pension share or alternativos volumes
   beyond what is captured.
